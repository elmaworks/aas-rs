# aas-rs

## Language Specification

- Rust (MSRV 1.78.0)

## About this repository

This is a project to translate an AAS library written in TypeScript into Rust.

## Quick Command

- `cargo build`: Build development configuration
- `cargo build --release`: Build release configuration
- `cargo fmt --all -- --check`: Run rustfmt for formatting codes
- `cargo clippy --fix --allow-dirty`: Run clippy for linting codes
